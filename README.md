# Vite + React 템플릿

### 스캐폴딩 명령

degit 명령을 사용해 프로젝트 스캐폴딩

```
degit Jisoo0907/vite-react <프로젝트 이름>
```

#### 폴더 구조

```
프로젝트 폴더
├── public
│   └── vite.svg
├── src
│   ├── main.jsx
│   └── styles
│      └── globals.css
├── .npmrc
├── App.jsx
├── eslint.config.js
├── index.html
├── jsconfig.json
├── package.json
├── pnpm-lock.yaml
├── prettier.config.cjs
├── README.md
└── vite.config.js
```
